VINTAGE CLOCK — EXPANDED CLOCK FACE GALLERY

This package adds these nine built-in clock faces:

- Railway Station
- Art Deco (1930s)
- Tiffany
- French Enamel
- Black Forest
- Skeleton Clock
- Steampunk
- Pocket Watch
- Vienna Regulator

INSTALLATION

1. Keep your existing three files in the faces folder:
   classic-cream.png
   aged-brass.png
   dark-victorian.png

2. Copy all nine new PNG files from this package into the same faces folder.

3. Replace your current index.html with the index.html in this package.

4. Upload/commit the changes to GitHub.

5. After GitHub Pages finishes publishing, refresh with Ctrl + F5.

The complete faces folder will contain:

classic-cream.png
aged-brass.png
dark-victorian.png
railway-station.png
art-deco.png
tiffany.png
french-enamel.png
black-forest.png
skeleton-clock.png
steampunk.png
pocket-watch.png
vienna-regulator.png
